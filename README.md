# 简单的基于有道词典的单词和短语翻译, 有音标
用法如下：

## 翻译
```
$ cd translate; go build
$ ./translate take up
英: [ˈteɪk ʌp]
美: [ˈteɪk ʌp]
开始从事（某项活动）：开始学习、练习或参与某项活动；
占用（时间、空间等）：使用或占据（时间、空间等）；
接受（挑战、任务等）：同意或接受（挑战、任务等）；
收起（衣物）：折叠或卷起（衣物）。
网络翻译: 占去; 开始从事; 占据; 拿起;
```

## 发音
```
$ cd pronounce; go build
$ ./pronounce take up
```
